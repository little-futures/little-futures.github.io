# About Little Futures

Little Futures is a shared digital garden maintained by Tom Critchlow and Brian Dell.

## How it works

- We both write notes in [Obsidian](https://obsidian.md)
- Notes are synced to a shared GitHub repository
- [Quartz](https://quartz.jzhao.xyz) builds the site and publishes to GitHub Pages
- Each note has frontmatter with an `author` field so you know who wrote what

## Structure

- **/notes** — shared notes where either of us can publish
- **/tom** — Tom's personal corner (notes, links, music, etc.)
- **/brian** — Brian's personal corner (notes, links, music, etc.)
- **/pages** — shared static pages like this one
